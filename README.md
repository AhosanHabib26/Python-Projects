Python Projects

A curated collection of practical Python applications, GUI games, and automation scripts. This repository showcases various implementations of Python, ranging from command-line interfaces (CLI) to graphical user interfaces (GUI) using `tkinter`.



📂 Project Showcase

| Number| Project Name | Description | Interface | Core Libraries |

🛠️ Detailed Project Overview

 1. Digital Clock Widget

A highly customized graphical desktop clock designed to mirror a dark-mode iOS widget interface. It dynamically formats time and structural dates cleanly using background thread execution triggers.

 2. Rent & Utility Splitter

A practical calculation tool targeting co-living arrangements. It aggregates standard base rents, shared food delivery charges, and calculated electricity tallies based on modular unit counters before distributing identical pricing across active roommates.

 3. Rock Paper Scissors Game

An interactive command-line application translating user-submitted system moves against random, algorithmic choices generated natively by python's core pseudorandom number utilities.

 4. Tic Tac Toe (Vs. Computer)

A full graphical grid matrix built using dynamic layout rendering. Features intelligent button state tracking, dynamic line highlights matching successful execution matrices, and an integrated, delayed computer move mechanism mimicking realistic system play tempos.

 🚀 Getting Started

 Prerequisites

Make sure you have Python 3.8 or higher installed on your computer. You can check your version by running:

```bash
python --version

```

 Installation & Execution

1. Clone this repository directly onto your local machine:
```bash
git clone https://github.com/AhosanHabib26/Python-Projects.git

```


2. Change directories into your cloned workspace:
```bash
cd Python-Projects

```


3. Execute any script instantly via your system terminal:
```bash
python "1.Build a Digital Clock.py"

```
